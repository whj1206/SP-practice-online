HTML tags reference
  • create a link: <a href="website">
  • create a list:
    <ul>unordered list</ul>
    <ol>ordered list</ol>
    the elements should be: <li>content</li>
  • start a new line: <br>
  • start a blank:   ;
  • insert a image:<img src="site on Internet"/>
    if you download the image to the folder, you can write its file name directly.

  • create a table:
    <table>
    border="n": border is the width of the line, default is 0
    width="m": the width of the table
    cellpadding="k": the height of the table
        <tr> is a row
            <td> is a column
            // write the content here //
            </td>
        </tr>
    </table> example:
    01 02 03
    11 12 13
